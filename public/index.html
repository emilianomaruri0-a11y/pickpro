Mis datos

Politicas de privacidad

PickPro guarda solo los datos necesarios para iniciar sesion, proteger la cuenta y personalizar la experiencia: nombre, usuario, correo o telefono, formato de momios y fuente de cuotas preferida.

Las contrasenas se almacenan con hash scrypt y las sesiones usan cookies HttpOnly con proteccion CSRF. El usuario, correo y telefono se mantienen unicos para evitar confusiones entre cuentas.

No vendemos datos personales ni compartimos informacion de cuenta con plataformas externas. Las cuotas y datos deportivos se usan solo para generar probabilidades, analisis y alertas dentro de la app.

Las predicciones son estimaciones probabilisticas. Ninguna senal, combinacion o analisis garantiza resultados, ganancias o aciertos seguros.
